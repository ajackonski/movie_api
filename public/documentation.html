Welcome to Myflix!

This is Myflix! an API that will allow users to access information about different movies, genres, and directors. The user will be able to sign up, update their personal information, and create a list of their favorite movies.

API Endpoint Documentation

Endpoint URL Method Description Format of Request Data Format of Response Data
/movies GET Fetches all movies None A JSON object conatining data bout all movies
/movies/:title GET Fetch data on a single movie by title None A JSON object containing data bout the requested movie
/movies/genre/:genreName GET Fetch data about a genre by name None A JSON object containing data about the requested genre
/movies/director/:directorName GET Fetch data about a director by name None A JSON object containing data about the requested director
/users POST Create a new user A JSON object holding data from the user to be added ex: { "name": "Alex", "email": "Alex@email.com" } A JSON object holding data from the user that has been added plus uuid ex: { "fullname": "Alex", "email": "Alex@email.com", "id": "200f45b2-d144-49a6-9252-863c0fec99a9" }
/users/:id PUT Update a user entry by ID A JSON object containing data from the user that is to be updated: { "name": "New Alex", "email": "NewAlex@email.com" } Text confirming the user entry has been updated
/users/:id/:movieTitle POST Adds a new movie to a user's favorite movie list A JSON object containing data to be updated to the users favorite movie list: { "Title": "Starship Troopers", "Director": "Paul Verhoeven", "Genre": { "Name": "Sci-fi" } } Text confirming the movie has been added to the requested users favorites list
/users/:id/:movieTitle DELETE Remove a movie from a user's favorite movie list None Text confirming the change to the users favorites list
/users/:id DELETE Unregister a user N/A Text confirming the user has been unregistered
